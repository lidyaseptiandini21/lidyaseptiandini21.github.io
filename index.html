PEMBUATAN GITHUB PAGES MENGGUNAKAN SISTEM GIT

GitHub Pages adalah layanan hosting gratis langsung dari repository yang ada pada github yang meng-host halaman statis web. Halaman statis tersebut berupa file dengan eksistensi HTML, CSS, JavaScript, dan lain sebagainya. GitHub pages dapat menampilkan web yang telah kita buat dengan alamat yang unik. Berikut adalah langkah-langkah pembuatan github pages menggunakan sistem git:

1. Pastikan anda memiliki akun github dan telah menginstall git pada laptop andaJika belum menginstall aplikasi git, maka anda dapat mendownload terlebih dahulu. Dan jika anda juga belum mempunyai akun github, anda dapat mendaftar terlebih dahulu.

2. Jika sudah memiliki akun github, anda dapat login dan buat Repository baru dengan nama (username anda).github.io.

3. Lalu, click create repository

4. Kemudian buka folder yang berisikan file yang akan kita upload ke repository github, lalu click kanan, setelah itu click Git Base Here.

5. Kemudian ketik “git init” pada git bash here untuk membuat local git repository.

6. Kemudian ketik “git add .” untuk menambahkan semua file yang ada pada folder ke staging area.

7. Kemudian ketik “git commit -m “first commit”” untuk memindahkan file yang berada di staging area ke repository. Git mencatat perubahan yang dimasukkan ke repository dengan memberi pesan “first commit”.

8. Kemudian ketik “git branch -M main" untuk membuat branch baru dengan nama “main”.

9. Kemidian ketikan “git remote add origin https://github.com/lidyaseptiandini21/lidyaseptiandini21.github.io.git”.

10.Kemudian ketik “git push -u origin main” untuk melakukan push local git repository ke github repository yang sebelumnya telah dibuat.

11.Setelah itu, kembali ke github repository yang telah dibuat lalu refresh browser anda. Jika semua file yang kita masukkan pada github repository sudah terlihat maka local git repository telah berhasil di-clone ke github repository. Jika pada halaman repository posisi kanan bawah terdapat keterangan github-pages active, maka hal ini berarti website anda sudah dapat diakses.

12.Kemudian, cek website anda dengan alamat sama dengan nama github repository yang telah dibuat sebelumnya yaitu (username anda).github.io di browser anda. congratulations you already have your own website :)
